Here is what I have learnt so far and how CSS is important.
CSS means Cascading Style Sheets. It is used for website layout in design and used alongside HTML.
In other words, CSS is a styling language which helps to control the appearance of a web page (whose structure has been built with HTML code).
CSS is important to make the website's page look exactly how we want it to look, in terms of colors, text sizes, margin, padding, borders, links appearance, hover colors, cursor style,
elements positioning, background images, screen sizes display, among many other styling options.

Explaining the difference between paddings, margins and borders.
Padding refers to the inner space between the element content and the element border.
It is useful to create a breathing-space appearance for the items of HTML elements.
Margin defines the outer space around elements, on the outside of any defined borders.
Its Properties include margin-top, margin-right, margin-bottom and margin-left).
Negative values are allowed for margin (e.g "margin: -30px auto;").
Border defines the lines on the four sides (top, right, bottom, left) around an element.
It has specific properties (border-style, border-color, border-width and border-radius) which are used to specify the style, color and size of the border of an element.
